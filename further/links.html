# Quick link

# 百科类

  • MDN (opens new window): Mozilla 维护的开发文档, 可能是最齐全的web文档
  • caniuse (opens new window): css 最好的兼容性百科
  • statcounter (opens new window): 查看世界浏览器市场份额

# 动画类

  • css动画库 (opens new window)
  • stylie (opens new window): css3贝塞尔曲线可视化工具

# 性能相关

  • webpagetest: 网页性能测试
  • jsperf (opens new window): 可以在不同的浏览器测试 JavaScript 的性能
  • csstriggers (opens new window): 一个CSS属性影响渲染的参考表(回流与重绘)

# Online tools

  • 草料二维码 (opens new window)
  • tinypng (opens new window): 压缩图片
  • 在线工具 (opens new window): 在线加解密, 转base64等
  • jsbin (opens new window): 在线编辑器, 可以将自己的代码保存发给别人
  • 可视化正则表达式 (opens new window) 将正则表达式转为可视化图表, 让你一眼能看懂这段正则是干啥的
